# InsighTA

## Problem Statement
Users often struggle to quickly understand trends and insights in a topic without reading long articles or reports.

## Solution
InsighTA is an AI-powered web platform that helps users discover key insights and trends in a simple and visual way using voice and text-based inputs.

## Features
- Topic-based insight discovery
- Trend analysis and summaries
- Clean and responsive UI
- AI-assisted content understanding

## Tech Stack
- Lovable AI (AI-assisted development)
- Firebase Hosting
- Web Technologies (HTML, CSS, JavaScript)

## AI Tools Used
This project was built using Lovable AI for rapid prototyping and frontend development.
